
# etch-a-sketch

create a webpage with a 16x16 grid of square divs
create the divs using javascript
container div can be written in html file
use flex-box to make the divs appear as a grid
set up a hover effect so that the grid divs change color when your mouse passes over them leaving a pixelated trail like a pen would
  hover happens on mouse enter and ends on mouse leave
colour change can be new class or background colour in javascript
button on top of the screen that will send a pop up asking for the number of squares per side for the new grid.
once entered old grid is removed and new grid should be generated in the same total space as before
limit user input to maximum of 100

extra credit
rather than squares being the same color randomize the squares rgb value each interaction
implement a progressive darkening effect where each interation darkens the square by 10%
  fully black square after 10 interactions
